## Topics Covered:-

   >Basic output statements.
# Operators.
  >Arithmetic Operators(+ - * / %).
  >Unary Operator(++ --).
  >Assignment Operator(=).
  >Comparison Operator(>= <= ==).
# Template Literals.
  >String interpolation using backticks.
  >Dynamic expressions inside strings.
# Conditional Statements.
  >if statement.
  >if-else statement.
  >else-if ladder.
  >Nested conditions.
  >Practice-based conditions.
# Logical Operator.
  >AND(&&).
  >OR(||).
# Truthy & False Values.
  >Understanding truthy and falsy conditions.
  >Real-world conditional checks.
# Switch Case.
  >Basic switch case.
  >Default case.
# Alert & Prompt.
  >alert() for user notification.
  >prompt() for taking user input.
